Developing apps for QuickSchools

Any developer can develop apps for QuickSchools. We provide a framework and tools for you to quickly build apps. The look and feel of your apps will generally mirror those of the built-in QuickSchools modules. Your apps can extend QuickSchools in two ways:

  • Add menus with your own code running under each menu, displaying content exactly as you specify.
  • Hook into various "insertion points" within the QuickSchools application so that you can extend our existing functionality.

Generally, the steps for creating an App is as follows:

  • Sign up for a trial QuickSchools account. This is free for 30 days.
  • Get familiar with turning on and off different features, and configuring those features.
  • Under the Features menu, turn on the App Development option.
  • You'll see a new App Developer Console menu.
  • Create an App.
  • Configure the menus that it will have.
  • Create the configuration screen that users will see when they turn on the app.
  • Start writing JavaScript to programmatically render screens.
  • Develop components which are made of HTML, CSS and JS when necessary. Instantiate these components from your JavaScript.
  • When you're ready, submit the application for approval.

Requirements Checklist

  • Chrome - We recommend developing only on Chrome at this time. Firefox has known issues related to debugging and code editing.
